1 최적화

1.1 최적화의 기본 요소


3가지 기본 요소

  1. 변수 (Decesion variable or unknown) \(\boldsymbol{x}\in \mathcal{M}_n(\mathbb{R})\)
  2. 목적함수 (objective function) \(f:\mathcal{M}_n(\mathbb{R}) \to \mathbb{R}\)
  3. 제약 조건 (Constraint) \(g_i:\mathcal{M}_n(\mathbb{R}) \to \mathbb{R}\), \(i=1,\ldots,\,m\) 에 대해 \(g_i(\boldsymbol{x})\le 0\) and/or \(h_j:\mathcal{M}_n(\mathbb{R}) \to \mathbb{R},\, j=1,\ldots,\,k\) 에 대해 \(h_j(\boldsymbol{x})=0\)


과정

  1. 모델링 : 목적함수, 변수, 제약조건을 찾고 확인한다.
  2. 모델을 만든 후 최적화 알고리즘을 사용하여 해를 구한다.


1.2 최적화의 수학 모델

수학적 표준 모델

  • 제약조건을 만족하는 \(\mathcal{C}\subset \mathcal{M}_n(\mathbb{R})\) 를 feasible region 이라고 한다. 즉 \(\mathcal{C}\) 는 아래와 같다. \[ \mathcal{C} = \{\boldsymbol{x}\in \mathcal{M}_n(\mathbb{R}) : g_i(\boldsymbol{x})\le 0,\,i=1,\ldots,\,m\}. \]

  • \(\displaystyle \boldsymbol{x}^\ast = \arg \min_{\boldsymbol{x}\in \mathcal{C}} f(\boldsymbol{x})\) 를 찾는다. 이 \(\boldsymbol{x}^\ast\) 를 최적해 (optimal solution) 이라고 한다.

  • \(g_i,\,h_j\) 와 같은 제약조건이 없다면 unconstrained 라고 하며, 그렇지 않다면 constrained 라고 한다.


등가 변환

  • 우리가 구하고자 하는 문제가 \(f(\boldsymbol{x})\) 를 최소화 하는 것이 아닌 최대로 하는 \(\boldsymbol{x}\) 를 찾는 문제라면 \(-f(\boldsymbol{x})\) 를 최소화 하는 문제이다.

  • constraint 가 \(g_i(\boldsymbol{x}) \ge 0\) 이라면 \(-g_i(\boldsymbol{x})\le 0\) 으로 바꿀 수 있다. \(g_i (\boldsymbol{x})=0\) 이라면 \(g_i(\boldsymbol{x}) \le 0\) 와 \(-g_i(\boldsymbol{x})\le 0\) 을 동시에 만족하는 조건으로 바꿀 수 있다.

  • 즉 수학적 표준 모델은 상당히 표면적으로 보이는 것보다 훨씬 넓은 범위의 문제를 포괄한다.


1.3 볼록 최적화 (Convex optimization)

convex function

\(f:\mathcal{M}_n(\mathbb{R})\to\mathbb{R}\) 이 다음을 만족하면 convect function 이라 한다. \[ \forall \theta\in [0,\,1],\, \boldsymbol{x},\, \boldsymbol{y}\in \mathcal{M}_n(\mathbb{R}) \implies f(\theta \boldsymbol{x}+(1-\theta)\boldsymbol{y}) \le \theta f(\boldsymbol{x}) + (1-\theta)f(\boldsymbol{y}) \]

convex set

\(A\subset\mathcal{M}_n(\mathbb{R})\) 이 다음을 만족하면 convex set 이라 한다.

\[ \forall \theta\in [0,\,1],\, \boldsymbol{x},\, \boldsymbol{y}\in A \implies \theta \boldsymbol{x}+(1-\theta)\boldsymbol{y}\in A \]


  • 목적함수가 convex function 이고 feasible region 이 convex set 인 경우의 최적화를 convex optimization 이라고 한다.
  • Convex optimization 의 모든 국소적 해는 전역적 해이다.
  • CVX (Matlab), CVXPY (Python)


2 Unconstrained convex optimizaiton 문제

  • \(\nabla_\boldsymbol{x}f(\boldsymbol{x}^\ast)=\boldsymbol{0}\) 인 \(\boldsymbol{x}^\ast\) 를 찾는 문제이다.

\[ \nabla_\boldsymbol{x} f = \begin{bmatrix} \dfrac{\partial f}{\partial x_1} & \cdots & \dfrac{\partial f}{\partial x_n}\end{bmatrix} \tag{1}\]


2.1 Gradient

\(\boldsymbol{x} = \begin{bmatrix}x_1 & \cdots & x_n \end{bmatrix}^T \in \mathcal{M}_n(\mathbb{R})\), \(\boldsymbol{a} \in \mathcal{M}_n(\mathbb{R})\), \(\boldsymbol{A}=\mathcal{M}_{m \times n}(\mathbb{R})\) 일 때 \(\boldsymbol{x}\) 에 대한 스칼라 함수의 gradient 는 다음과 같다.

\[ \begin{aligned} \nabla_\boldsymbol{x} (\boldsymbol{a}^T\boldsymbol{x}) &= \boldsymbol{a}^T\\[0.3em] \nabla_\boldsymbol{x} (\boldsymbol{x}^T\boldsymbol{a}) &= \boldsymbol{a}^T\\[0.3em] \nabla_\boldsymbol{x} (\boldsymbol{x}^T\boldsymbol{x}) &= 2\boldsymbol{x}^T\\[0.3em] \nabla_{\boldsymbol{x}} (\boldsymbol{x}^T\boldsymbol{Ax}) &= \boldsymbol{x}^T(\boldsymbol{A}^T +\boldsymbol{A}) \end{aligned} \tag{2}\]


2.2 직접법

  • \(\nabla_\boldsymbol{x}f(\boldsymbol{x}^\ast)=\boldsymbol{0}^T\) 인 \(\boldsymbol{x}^\ast\) 를 직접 구한다.


이제 몇가지 형태의 convex 함수인 목적함수에 대한 그래디언트 \(\nabla_{\boldsymbol{x}}f(\boldsymbol{x})\) 와 헤시안 행렬 \(\boldsymbol{H}_f\) 를 구해보자.

예제 1 (Affine 형태의 목적함수) \(f(\boldsymbol{x}) = \boldsymbol{a}^T\boldsymbol{x} + \boldsymbol{b}\) 인 경우

\[ \nabla_\boldsymbol{x}f(\boldsymbol{x})=\boldsymbol{a}^T,\qquad \boldsymbol{H}_f = \boldsymbol{0}. \]

이다. 따라서 \(\boldsymbol{a}^T=\boldsymbol{0}^T\) 인 특별한 경우가 아니면 최소값이 존재하지 않는다.


예제 2 (Quadratic 형태의 목적함수) 대칭행렬 \(\boldsymbol{P}\) 에 대해 \(f(\boldsymbol{x}) = \boldsymbol{x}^T\boldsymbol{Px} + \boldsymbol{b}^T\boldsymbol{x}+c\) 인 경우

\[ \nabla_{\boldsymbol{x}}f(\boldsymbol{x})= 2\boldsymbol{x}^T\boldsymbol{P} + \boldsymbol{b}^T,\qquad \boldsymbol{H}_f = 2\boldsymbol{P} \]

이다.


예제 3 (제곱 형태의 목적함수) 대칭행렬 \(\boldsymbol{P}\) 에 대해 \(f(\boldsymbol{x}) = \|\boldsymbol{Ax}-\boldsymbol{b}\|^2\) 인 경우

\[ f(\boldsymbol{x})= \boldsymbol{x}^T\boldsymbol{A}^T\boldsymbol{Ax} - \boldsymbol{x}^T\boldsymbol{A}^T\boldsymbol{b}-\boldsymbol{b}^T\boldsymbol{Ax}+\boldsymbol{b}^T\boldsymbol{b} \]

이므로 gradient 와 헤시안 행렬은 다음과 같다.

\[ \nabla_{\boldsymbol{x}}f(\boldsymbol{x})= 2\boldsymbol{x}^T\boldsymbol{A}^T\boldsymbol{A} -2\boldsymbol{b}^T\boldsymbol{A},\qquad \boldsymbol{H}_f = 2\boldsymbol{A}^T\boldsymbol{A}. \]

이 때 \(\boldsymbol{x}^\ast = (\boldsymbol{A}^T\boldsymbol{A})^{-1}\boldsymbol{A}^T\boldsymbol{b}\) 이어야 한다. 이 때 \((\boldsymbol{A}^T\boldsymbol{A})^{-1}\boldsymbol{A}^T\) 는 잘 알려진 무어-펜로즈 좌측 유사역행렬(left pseudoinverse matrix) 이다.


2.3 반복법

  • Gradient descent : \(\boldsymbol{x}_{k+1} = \boldsymbol{x}_k -\alpha_k \nabla_\boldsymbol{x} f(\boldsymbol{x}_k)\). \(\alpha_k\) 는 내부적으로 정해지는 양의 scalar.
